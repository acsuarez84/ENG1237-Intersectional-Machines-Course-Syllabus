Section 10

EXAMPLE of AGENTIC Textual Analysis

Live example projects — the instructor’s own demos — so you can see what agentic, AI-assisted textual analysis looks like before you build your own.

These are working demonstrations built with the same tools you will use this semester—Claude, Claude Code, GitHub, and ChatGPT. Open any of them to see how a prompt journal, a distant reading, or a textual map can become a finished, shareable artifact. Start with the textual map examples—that is the first artifact you will make (see section 9).
